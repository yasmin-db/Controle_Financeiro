💰 FinTrack Bot: Controle Financeiro Inteligente via WhatsApp

Projeto de Trabalho de Conclusão de Curso (TCC)
Um ecossistema completo para gestão de finanças pessoais que integra a praticidade das mensagens instantâneas com o poder de análise de dados de um dashboard web.

🎯 Ideia e Objetivo

Muitas pessoas desistem de controlar suas finanças devido à complexidade de abrir aplicativos bancários ou preencher planilhas manualmente no final do dia.

O FinTrack Bot resolve essa barreira permitindo que o usuário registre seus gastos no exato momento em que eles ocorrem, utilizando uma interface simples e familiar: o WhatsApp.

🔄 Fluxo do Projeto
Entrada
O usuário envia uma mensagem rápida:
Jantar; 55.90; Alimentação
Processamento
Um bot desenvolvido em Node.js + Baileys:
interpreta a mensagem
valida os dados
solicita confirmação
Persistência
Após confirmação, os dados são armazenados em um banco de dados SQL.
Visualização
Um dashboard web responsivo consome esses dados e gera:
gráficos mensais
análises anuais
evolução financeira
